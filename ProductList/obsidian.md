# Obsidian — 本地知识管理工具

## 📌 简介

| 项目 | 内容 |
|------|------|
| **开发者** | Shida Li 和 Erica Xu（加拿大小团队） |
| **首次发布** | 2020 年 |
| **用途** | 本地优先的 Markdown 笔记应用，构建个人知识库 |
| ** slogan** | Your Second Brain（你的第二大脑） |

## 🎯 用来做什么？

- **记笔记**：纯 Markdown 格式，没有厂商锁定
- **知识管理**：通过笔记间的双向链接，构建知识网络
- **个人知识库**：把你的笔记变成一个可搜索、可关联的知识系统
- **日记/周记**：支持模板，适合每日记录
- **项目管理**：用 Kanban、看板、时间线等插件
- **写作**：专注模式、实时预览，适合写长文

## 🏗️ 核心概念

### 1. 笔记库（Vault）
> 一个文件夹，里面所有的 Markdown 文件就是你的笔记库。
> 你的笔记库就在：`/Users/mk/obsidian_files/obs_repository/`

**特点**：Obsidian 不存数据到云端，所有笔记就是本地文件。你可以用 iCloud、Dropbox、Git 来同步备份。

### 2. 双向链接（[[ 双链 ]]）
这是 Obsidian 最核心的功能：

```markdown
[[Python 基础]]        → 链接到另一篇笔记
[[Python 基础|查看]]  → 修改显示文字
```

链接后，笔记底部会自动显示"反向链接"——哪些笔记引用了当前笔记。

### 3. 图谱视图（Graph View）
> 点击右侧的「打开图谱视图」按钮，可以看到笔记间的链接关系图。
> 一个节点 = 一篇笔记，一条线 = 一次链接。

### 4. 标签（Tags）
```markdown
#重要     #AI/学习     #项目/前端
```
标签支持层级，`#AI/学习` 和 `#AI/工具` 会归类在 `AI` 下面。

## 🔧 常用操作

### 快捷键（macOS）

| 操作 | 快捷键 |
|------|--------|
| 新建笔记 | `Cmd + N` |
| 搜索 | `Cmd + Shift + F`（全局搜索） |
| 打开快速切换 | `Cmd + O`（像 Mac 的 Spotlight） |
| 打开命令面板 | `Cmd + P`（所有操作都能搜到） |
| 添加标签 | `Cmd + T` |
| 插入链接 | `Cmd + K` |
| 粗体 | `Cmd + B` |
| 斜体 | `Cmd + I` |
| 预览/编辑切换 | `Cmd + E` |
| 缩进 | `Tab` |
| 取消缩进 | `Shift + Tab` |

### 基础操作流程
```
创建笔记 → 写内容 → 用 [[ ]] 链接相关笔记 → 打 #标签 → 图谱查看
```

## 💡 日常使用示例

### 示例 1：用 Obsidian 做学习笔记
1. `Cmd + N` 新建笔记，名字叫"Python 列表详解"
2. 写笔记内容
3. 在笔记中链接触及相关笔记：`[ [Python 基础] ]`
4. 添加标签：`#Python/基础`
5. 以后复习时，点开图谱视图就能看到知识连接

### 示例 2：用日记模板
```
设置 → 核心插件 → 日记 → 设置日期格式和模板位置
```
每日点击左侧日历的日期，自动生成当日日记，模板可以预设好结构：
```markdown
# {{date}}

## 今日完成
- 

## 今日收获
- 

## 明日计划
- 
```

### 示例 3：建立个人知识库
```
├── 项目/
│   ├── AI 资讯日报/
│   └── Hermes-folder/
├── 学习/
│   ├── Python 基础.md
│   ├── Git 入门.md
│   └── SQL 查询技巧.md
├── 日记/
│   └── 2026-05-12.md
└── 模板/
    └── 日报模板.md
```

## 🧩 推荐插件（社区插件）

| 插件 | 用途 |
|------|------|
| **Dataview** | 把笔记当数据库查询，自动生成目录/列表 |
| **Templater** | 高级模板引擎，自动插入日期、时间 |
| **Calendar** | 日历视图，方便跳转日记 |
| **Kanban** | 看板视图，适合项目管理 |
| **Excalidraw** | 手绘风格画图，画流程图和白板 |
| **Paste URL into Selection** | 选中文字后粘贴链接，自动生成链接 |
| **Image Toolkit** | 更好的图片浏览体验 |

> 安装：设置 → 社区插件 → 浏览 → 搜索插件名 → 安装并启用

## 🚨 常见问题

| 问题 | 解决 |
|------|------|
| 忘记快捷键？ | `Cmd + P` 打开命令面板，搜你想做的事 |
| 笔记找不到了？ | `Cmd + O` 快速跳转，或 `Cmd + Shift + F` 全文搜索 |
| 想导出为 PDF？ | 更多菜单 → 导出为 PDF |
| 想在手机和电脑间同步？ | 官方同步（收费）或 iCloud/Dropbox 免费方案 |
| 插件不生效？ | 检查是否已启用（设置 → 社区插件） |

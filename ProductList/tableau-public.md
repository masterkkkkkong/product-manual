# Tableau Public — 免费数据可视化平台

## 📌 简介

| 项目 | 内容 |
|------|------|
| 开发者 | Tableau Software（现属 Salesforce） |
| 首次发布 | 2010 年 |
| 用途 | 免费的拖拽式数据可视化工具，创建交互式图表和仪表板并发布到 Web |

## 🎯 用来做什么？

- **快速制作数据可视化**：拖拽字段即可生成柱状图、折线图、地图、散点图等 20+ 种图表
- **创建交互式仪表板**：组合多个图表、添加筛选器，让读者自己探索数据
- **发布到公共网络**：一键发布到 `public.tableau.com`，任何人均可在线查看
- **学习数据分析**：免费版本功能完整，是入门数据可视化的最佳选择之一

## 🔧 常用功能

> Tableau Public 是桌面 GUI 应用，没有命令行工具（Tableau Desktop 付费版有 `tabcmd` 命令行）。

```bash
# macOS：从 https://public.tableau.com 下载安装
# 安装后应用名称为 "Tableau Public.app"
# 打开命令（如果已安装）
open /Applications/Tableau\ Public.app
```

### GUI 核心操作速查

| 操作 | 说明 |
|------|------|
| 连接数据源 | 支持 Excel、CSV、Google Sheets、文本文件、PDF |
| 拖拽字段到列/行 | 列 = X 轴，行 = Y 轴，自动生成图表 |
| 标记卡 | 调整颜色、大小、标签、详细信息 |
| 筛选器 | 拖拽到筛选器区，过滤不想显示的数据 |
| 仪表板 | 组装多个工作表，添加交互筛选器 |
| 发布 | 登录账户后，菜单 → 保存到 Tableau Public → 生成公开链接 |

## 💡 日常使用示例

### 示例 1：从 CSV 文件创建柱状图

1. 打开 Tableau Public，点击 **连接 → 文本文件**
2. 选择你的 CSV 文件（例如销售额数据）
3. 在 **数据源** 页面确认字段类型正确
4. 点击底部 **工作表 1** 标签进入制作界面
5. 把「月份」字段拖到 **列**，把「销售额」字段拖到 **行**
6. 自动生成柱状图，在右侧 **标记** 卡中可调整颜色
7. 点击顶部保存按钮 → **保存到 Tableau Public** → 登录后即可发布

### 示例 2：创建带筛选器的交互式仪表板

假设已做好两个工作表：「月度销售额柱状图」和「各区域销售额饼图」：

1. 点击底部 **新建仪表板** 标签
2. 将两个工作表从左侧拖到仪表板区
3. 点击左侧 **显示标题** 为仪表板起名字
4. 把「年份」字段拖到仪表板空白处，自动变成筛选器
5. 点击筛选器右上角的 ▼ → **应用到工作表 → 全部使用此数据源**
6. 现在点击不同年份，两个图表会同步更新
7. **保存到 Tableau Public** 发布，复制链接分享给他人

### 示例 3：制作世界地图

1. 准备一个包含国家名和数值的 Excel 文件
2. 导入后，双击「国家/地区」字段
3. Tableau 自动识别地理角色，生成世界地图
4. 把数值字段拖到 **颜色** 标记，生成着色地图
5. 把数值拖到 **大小** 标记，生成气泡地图
6. 把国家名拖到 **标签**，显示国家名称

## 🚨 常见坑点

| 问题 | 原因 | 解决方法 |
|------|------|----------|
| 发布时提示需要登录 | Tableau Public 必须登录账户才能发布 | 免费注册一个 Tableau Public 账户 |
| 地图显示「未知」 | 城市名称拼写或格式不被识别 | 检查列的数据类型是否为「地理角色」，使用标准英文名称 |
| 数据超过 1000 万行 | Tableau Public 免费版限制 | 汇总数据或在 Excel 中先聚合 |
| 工作簿显示乱码 | CSV 文件编码问题 | 另存为 UTF-8 编码的 CSV 或用 Excel 连接 |
| 图表无法发布 | 工作簿包含敏感数据（Public 意味着公开） | 确认数据可公开，或改用付费的 Tableau Desktop |
| 筛选器只控制一个图表 | 筛选器未应用到所有工作表 | 右键筛选器 → 应用到工作表 → 选择「全部」 |
